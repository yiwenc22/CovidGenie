# CovidGenie
Build it on your local device. <br/>
Open your Commed prompt and follow the steps <br/>
Step 1 : ```npm install -g expo-cli```  <br/>
Step 2 : ```cd into the folder```       <br/>
Step 3 : ```npm start```                <br/>
Than you can see it on Web, phone by scaning the QR code. <br/>
